# Chapter 02: Variables for Strings - JavaScript Exercises

## Project Overview

This directory contains JavaScript exercises for **Chapter 02: Variables for Strings** from a JavaScript learning curriculum. Each subdirectory (`question-01` through `question-09`) contains a standalone exercise focused on practicing variable declaration, string assignment, and basic output methods in JavaScript.

### Technologies Used
- **JavaScript (ES3/ES5)**: Uses `var` for variable declarations
- **HTML**: Simple HTML scaffolding to run JavaScript in the browser
- **Browser APIs**: `alert()`, `document.writeln()` for output

---

## Exercise Summaries

| Question | Topic | Description |
|----------|-------|-------------|
| **01** | Variable Declaration | Declare a variable `username` without assignment |
| **02** | Variable Assignment | Declare `myName` and assign a full name string |
| **03** | Alert with Variable | Store "Hello World" in a variable and display via `alert()` |
| **04** | Multiple Variables | Store student name, age, and course; display each with alerts |
| **05** | Newline Characters | Create a PIZZA pattern using `\n` for line breaks |
| **06** | String Concatenation | Concatenate email variable into a message string |
| **07** | Book Name Alert | Store book title and display with prefix text |
| **08** | document.writeln() | Write HTML content directly to the document |
| **09** | Special Characters | Display a decorative banner using Unicode characters |

## Usage

### Running an Exercise

1. Open any `index.html` file in a web browser
2. The associated `app.js` will execute automatically
3. Observe the output (alerts or document content)

**Example:**
```bash
# Open question-01 in default browser (Windows)
start question-01\index.html
```

### Testing All Exercises

Open each `index.html` file sequentially to verify all solutions work correctly.

## Coding Conventions Observed

1. **Variable Declaration**: Uses `var` (ES3/ES5 style, typical for beginner JavaScript courses)
2. **Naming**: Descriptive variable names (`studentName`, `studentAge`, `email`)
3. **Output Methods**: 
   - `alert()` for popup messages
   - `document.writeln()` for writing to the document
4. **String Operations**: Concatenation with `+`, newline characters with `\n`
5. **File Structure**: Each question is self-contained in its own folder

---

## Learning Objectives

By completing these exercises, learners should understand:
- How to declare variables using `var`
- How to assign string values to variables
- How to display variable content using `alert()`
- How to concatenate strings
- How to use special characters (e.g., `\n` for newlines)
- How to write content to the HTML document via JavaScript
